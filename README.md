# ProyectoDrones

Monorepo que integra los componentes principales del proyecto de drones.

## Estructura

- **PDBackend_G-3** - API backend para gestión de drones
- **Frontend/V4** - Interfaz web para control de drones

## Clonar con submódulos

\`\`\`bash
git clone --recurse-submodules https://github.com/JoaquinARoca/PD-FullStack-G-3.git
\`\`\`

## Actualizar submódulos

\`\`\`bash
git submodule update --remote
\`\`\`
